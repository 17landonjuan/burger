Burger is a full stack application for anyone who loves themselves a good burger. The app allows the user to enter in what kind of burger they would like to eat. It will then add each burger submitted to a database, and create a list of all burgers submitted. Each burger will then be populated on the list with a button next to it to "Devour" the sweet juicy beef patties, topped with crisp lettuce, plump tomato slices, savory sweet onion and of course a slice of ooey-gooey melty cheese!
